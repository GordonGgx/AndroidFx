# SimilarAndroid
简化JavaFx的开发过程，提供方便使用的组件。

## Features
1. 提供SimilarPreloader
2. 提供SimilarApplication
3. 提供Activity支持
4. 异步任务支持，支持FX->work线程互相切换

## TODO
